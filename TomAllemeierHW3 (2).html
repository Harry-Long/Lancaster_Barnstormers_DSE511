import sqlite3
import pandas as pd
import seaborn as sns
import matplotlib.pyplot as plt
conn = sqlite3.connect('financial.db')

account = pd.read_sql_query("SELECT * FROM account", conn)
client = pd.read_sql_query("SELECT * FROM client", conn)
credit = pd.read_sql_query("SELECT * FROM card", conn)
disp = pd.read_sql_query("SELECT * FROM disp", conn)
trans = pd.read_sql_query("SELECT * FROM trans", conn)
ordr = pd.read_sql_query("SELECT * FROM 'order'", conn)
loan = pd.read_sql_query("SELECT * FROM loan", conn)
demo = pd.read_sql_query("SELECT * FROM district", conn)

conn.commit()
conn.close()

Now we can 'x-ray' the distributions a bit. There is a clear trend that smaller and shorter term loans are more likely to fall into the 'paid back in full' category (A). Loans in category 'on track' are more likely to be smaller amounts with longer durations. However, since these are scaled by the actual counts of each, still can't see much about our other two (much smaller) groups from the margins.
